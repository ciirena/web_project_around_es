# Around the US

## Descripcion

- Around the US es una página web donde se puede ver fotos de lugares de Estados Unidos, funciona como una pequeña red social para mostrar al usuario imagenes de otros perfiles.

## Funcionalidad

- Otorgar "me gusta" a las tarjetas, en caso de que al usuario le guste una foto en especial, un aspecto importante de las paginas web interactivas de redes sociales.
- Opcion de eliminar tarjetas en caso que el usuario ya no considere mantenerlas en el perfil.
- Tambien se pueden agregar nuevas imagenes con su titulo y descripcion.
- Funcion de dar un "click" en cada foto para poder aumentar su tamaño y poder observar la imagen en un tamaño mayor.
- Se puede abrir y cerrar modales, para agregar nuevas imagenes y cambiar el nombre del usuario.

## Tecnologias
